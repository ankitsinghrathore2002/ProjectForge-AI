# ⚒️ ProjectForge AI

> AI-Powered Software Solution Architect built with **Streamlit** and **Google Gemini**.

ProjectForge AI transforms a simple software idea into a complete software development blueprint by generating project documentation, system architecture, SQL database schemas, AI-powered project analysis, and blueprint reviews.

---

## 🚀 Features

- 📄 AI Software Blueprint Generator
- 🏗️ System Architecture Generator
- 🗄️ SQL Database Schema Generator
- 📊 AI Project Analysis
- 🔍 AI Blueprint Review
- 📥 Export Blueprint as:
  - PDF
  - DOCX
  - Markdown
  - SQL
- 🎨 Modern Streamlit UI
- ⚡ Powered by Google Gemini AI

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/ankitsinghrathore2002/ProjectForge-AI.git
```

Move into the project directory

```bash
cd ProjectForge-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file and add your Google Gemini API key.

Run the application

```bash
streamlit run app.py
```
